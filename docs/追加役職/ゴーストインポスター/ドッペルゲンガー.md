# イグナイター

「第一回 究極の企画会  ～'Extreme' in your hands～」よりカデさん提案のもの

プレイヤーのダミーを生存者に見せることが出来るゴーストインポスター役職<br>
- ボタン能力「ドッペル」
  - ボタンを押すことでプレイヤー選択画面が開き、選択したプレイヤーのダミーを生存者に見せることが出来る
    - ダミーは全プレイヤーから視認することが出来る
      - フェイカーのダミーと違いアドミンを騙すことはできない
    - ダミーは壁も貫通します

- 能力発動後の会議メッセージ
  - 「誰かのドッペルゲンガーが発見されました」
